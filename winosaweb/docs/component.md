# Component Documentation

Dokumen ini berisi penjelasan singkat mengenai komponen utama yang digunakan pada frontend website Winosa. Struktur komponen dibuat reusable agar memudahkan pengembangan dan maintenance.

## Navbar
Navbar digunakan sebagai navigasi utama pada website. Komponen ini mendukung tampilan responsive, termasuk menu desktop dan mobile (hamburger menu).

Fitur utama:
- Navigasi antar halaman
- Pergantian bahasa
- Responsive menu untuk mobile
- Perubahan tampilan saat scroll

## Footer
Footer digunakan untuk menampilkan informasi tambahan di bagian bawah website.

Fungsi utama:
- Navigasi tambahan
- Informasi perusahaan
- Link ke sosial media (Instagram, LinkedIn, WhatsApp)
- Branding dan tagline

## Button
Button merupakan komponen reusable yang digunakan untuk berbagai aksi seperti submit form atau navigasi.

Karakteristik:
- Dapat digunakan di berbagai section
- Mendukung custom text
- Memiliki efek interaksi seperti hover dan click

## CTA Section
CTA (Call To Action) section digunakan untuk mengarahkan user melakukan aksi tertentu, seperti menghubungi atau subscribe newsletter.

Fitur:
- Form input email dengan validasi
- Integrasi API untuk newsletter
- Feedback sukses dan error
- A/B testing sederhana
- Tracking event menggunakan analytics

## Form Input
Digunakan pada bagian newsletter untuk menerima input dari user.

Fitur:
- Validasi email
- Feedback error dan success
- Styling minimal dan clean

## Layout Components
Beberapa komponen digunakan sebagai struktur dasar layout seperti container, section, dan grid.

Tujuan:
- Menjaga konsistensi layout
- Mempermudah pengaturan spacing dan alignment

## Animation / Microinteraction
Beberapa komponen memiliki efek interaksi untuk meningkatkan user experience.

Contoh:
- Hover effect pada button dan link
- Scale effect saat klik
- Fade-in saat scroll
- Transition yang halus

Semua animasi dibuat ringan agar tidak mengganggu performa website.
